### For Maximum Marks:
1. **Print/export** all three documents as PDFs for submission
2. **Include code files** as appendices or separate submission
3. **Take high-quality screenshots** showing key features
4. **Highlight algorithmic sections** in your code with comments

### Common IB Examiner Expectations:
- **Depth over breadth**: Better to explain fewer techniques thoroughly
- **Code literacy**: Demonstrate understanding of your own code
- **Professional practices**: Show awareness of industry standards
- **Problem-solving**: Explain how you solved specific challenges
- **Reflection**: Show understanding of why choices were made
